Để chạy project, mở bằng IntelliJ IDEA, 
                 mở thư mục dohoa_6, 
                 sử dụng jdk17 và them tất cả file có đuôi .jar trong thư mục lib trong freetts-1.2.2-bin\freetts-1.2 
                 nếu chưa tự động add, reload file maven trước khi chạy.
Ngoài ra nếu chạy bị lỗi ta xem lại các đường dẫn địa chỉ đến các file txt xem đã trùng khớp với trong code chưa
- Đường dẫn để đọc file txt trong code là: 
     C:\dohoa_6\src\main\java\com\example\dohoa\ dictionaries.txt
- Đường dẫn để ghi vào file txt trong code là: 
     C:\dohoa_6\src\main\java\com\example\dohoa\ dictionaries.txt
- Đường dẫn để lưu vào file txt trong code favourite là: 
     C:\dohoa_6\src\main\java\com\example\dohoa\ myFavorite.txt
Chạy tệp HelloApplication trong package GUI để chạy chương trình.
